- 狙い: 「いつもの店で、いつもと違う一杯を試した」生活の小さな変化を主役にする。
- 構図: 左下にテーブル上のグラス2個。奥にぼんやりしたメニュー片。右半分は光の当たったテーブルまたは壁でタイトル余白。
- 色: ミルクティーの橙茶、コーヒーブラウン、白タイル、淡いステンレスグレー。彩度は控えめ。
- モチーフ: Teh Tarik風の泡、Cham風の濃いアイスグラス、メニューの角、ストロー、結露。
- 文字の扱い: 画像内には文字なし。タイトルは後から日本語で重ねる想定。
- 参照元から離すための注意: 具体的な店舗、料理写真、人物、レシピ写真の注ぎ構図を避ける。ママックは抽象的な店内光とテーブル面だけで表す。

### 案2: Cham Cが少し軽く見えた

- 狙い: ノーマルChamよりCham Cが自分に合った、という個人的な味覚の輪郭を静かに表現する。
- 構図: 中央左に3つの飲み物を横並びまたは少しずらして配置。濃い茶、ミルクティー、軽いミルクブラウンの順に微妙な差を作る。右上に大きな余白。
- 色: コーヒーブラウンから淡いクリームブラウンへのグラデーション。ただし一色に寄せすぎず、タイルの白、ストローの赤茶、影の青灰を差す。
- モチーフ: 氷の透明感、泡の薄い層、グラスの縁、結露、小さなメニューカード。
- 文字の扱い: 飲料名のラベルは使わず、丸や短い線で「選択肢」の気配だけ出す。
- 参照元から離すための注意: 商品パッケージやEC写真のようにしない。味の比較図にもせず、あくまで生活の一場面にする。
