Media Hub

Podcast、YouTube、RSS、Substack、note、記事を、学習ログ・実験・発信・将来プロジェクト案につなげるための手動メモ置き場です。

Safety rules

本文、長い引用、音声、動画、transcript全文は保存しません。外部コンテンツは参照リンクとして扱い、短い自分のメモだけを残します。

Status definitions

  • Manual note: 自分で短くメモしたもの
  • AI-assisted draft: AI補助あり・未確認
  • Reviewed: 人間が原典確認済み

Date and usage defaults

  • Accessed date: YYYY-MM-DD
  • Published date: 不明な場合は Unknown
  • Usage note: Linked for reference. No full text reproduced.
