Meta Ads / GTM practice

計測まわりの練習メモ

このサイトは、GTMやMeta Pixelの設置・イベント確認を練習する場所として始まりました。

Current setup note

既存ページでは Google Tag Manager ID GTM-N787TD4T を使っています。v1でもこのIDを維持します。

GTM / Pixel setup notes

  • GTMのhead用scriptとbody直後のnoscriptをセットで確認する。
  • ページを追加したら、同じGTM IDが入っているか確認する。
  • Meta Pixelを使う場合は、GTM経由で発火条件を整理する。

Event examples

  • PageView: 各ページ表示時の基本イベント。
  • Lead: 問い合わせや登録完了ページで使う想定。
  • Custom event: 学習ログ閲覧、実験ページ閲覧などの練習用。

Debugging checklist

  • GTM Previewで対象ページに接続できるか。
  • Tag AssistantでGTMコンテナが検出されるか。
  • イベント名と発火条件がメモ通りか。
  • 本番URLとローカルURLを混同していないか。

Mistakes and fixes

  • 古いデモページの目的が分かりにくい場合は、v1ナビから外す。
  • 新規ページを作ったらGTMのnoscriptを入れ忘れていないか確認する。
  • 外部ツールで確認するときは、公開URLとファイルパスの違いに注意する。
